over HTML

HTML (Hypertext Markup Language) is de standaard taal die gebruikt wordt voor bijvoorbeeld: een website. hierin wordt gebruik gemaakt van bijvoorbeeld: images, videos, tabellen, formulieren, teksten, etc.

over CSS

CSS (Cascading Style Sheets) wordt gebruikt als een soort van styling van de website, hierin wordt gebruik gemaakt van background images, kleuren, en allemaal andere soorten dingen die je gebruikt voor het stylen van een website.
